Upload Patient 3D Scan

Upload the STL scan, fill in patient details, and tap Generate Link. A short link is sent to the patient's WhatsApp — they tap it and the 3D scan opens directly on their phone. Any file size. Any device.

⚙️ One-time Firebase setup required

To store large scan files and generate short links, this app uses Firebase (free). You only need to set this up once.

  1. Go to console.firebase.google.com → Create a project (free)
  2. Add a Web App → copy the firebaseConfig object
  3. Enable Firestore Database (test mode) and Storage (test mode)
  4. Open your index.html → paste the config at the top where it says PASTE_YOUR_…
  5. Push to GitHub → done! All links will now be short and work on any device.
